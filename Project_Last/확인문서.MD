완성된 이후에 알려줘

모델이 완성된 후에 rsi, macd지표가 변동에 따른 다음날 종가에 어떻게 반영된건지에 대한 결과도 따로 문서로 만들어놔. 





## 5. 사용자 시나리오

분석자는 정제된 최신 일별 데이터를 준비한 뒤 학습 또는 예측 명령을 실행한다.

시스템은 다음 작업을 수행한다.

1. 원본 데이터 검증 및 정제
2. 과거에서 미래 순서로 정렬
3. 파생 특성과 익일 목표값 생성
4. 시간 순서 기반 학습 및 검증
5. 기준 모델과 후보 모델의 성능 비교
6. 최종 모델, 설정, 평가 결과 저장
7. 최신 행을 기준으로 다음 거래일 시가·종가 예상치 출력
