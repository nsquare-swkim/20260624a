# 독립 테스트 평가 보고서

## 평가 원칙

검증 구간의 가격 MAE만으로 모델을 먼저 고정한 후, 2025-08-14부터 2026-05-29까지의 독립 테스트 191행을 한 번 평가했다.

- 익일 시가 선택 모델: 20일 평균 수익률
- 익일 종가 선택 모델: Naive
- 재학습 데이터: 2018-08-28 ~ 2025-08-12
- 테스트 경계 직전 목표값 누수 행 제외

이 테스트 구간은 이미 결과를 확인했으므로 앞으로 모델 선택이나 하이퍼파라미터 조정에 다시 사용하면 안 된다.

Naive 종가 모델은 항상 수익률 0을 예측한다. 따라서 방향 정확도 1.05%는 실제 수익률이 정확히 0인 날의 비율이며 상승·하락 예측 능력을 의미하지 않는다.

## 참고 모델 결과

테스트에서 시가는 Ridge가 MAE 3,384.57원으로 사전에 선택한 20일 평균 모델보다 우수했다. 종가는 20일 평균 모델이 MAE 4,554.75원으로 Naive보다 소폭 우수했다.

그러나 이 사실은 테스트 결과를 확인한 뒤 알게 된 것이므로 해당 모델로 교체하면 테스트 구간에 대한 사후 선택이 된다. 따라서 현재 테스트 성능의 공정한 결론은 검증에서 미리 선택한 모델 결과를 기준으로 한다.

## 결론

- 시가 방향 정확도 57.07%는 약한 방향성 신호가 있을 가능성을 보여주지만, 가격 오차가 크고 단일 테스트 기간 결과이므로 강한 신뢰를 줄 수 없다.
- 종가 선택 모델은 단순 기준선일 뿐 예측 신호가 없다.
- 두 모델 모두 투자 판단이나 자동매매에 사용할 수준이 아니다.
- 현 상태의 모델은 가격 범위를 대략 참고하는 연구용 기준선으로만 사용할 수 있다.
- 독립 테스트를 포함한 종합 신뢰도는 낮음으로 평가한다.
